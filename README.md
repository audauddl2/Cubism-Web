# Cubism-Web
* __FrontEnd:__ bootstrap(AdminLTE)
* __BackEnd:__ Spring(Java)

시각장애인을 위한 기억력 게임기 웹 페이지 개발

1. 게임 기록들을 구글 차트를 이용하여 시각화
2. 새로운 버전(게임)을 업로드
